📋 How to add a project

Easiest (when the site is served from a server):

  1. Fill out the form below and add at least one gallery image.
  2. Click "Add project & download projects.json".
  3. Replace your repo’s data/projects.json with the downloaded file, then commit and push.

Manual (e.g. opening this page as a file):

  1. Click "Generate JSON", then "Copy to Clipboard".
  2. Open data/projects.json and paste the new block into the projects array (after the last project, add a comma then paste).
  3. Save and push.

Tip: Run npx http-server in the project folder and open /add-project.html to use the download option.
